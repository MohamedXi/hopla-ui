# Button

Le composant Button est utilisé pour déclencher une action ou un événement, comme soumettre un formulaire, ouvrir une boîte de dialogue, annuler une action, ou effectuer une opération de suppression.

## Importation

```jsx
import { Button } from 'hopla-ui';
```

## Utilisation

```jsx
<Button variant="primary">Cliquez-moi</Button>
```

<div className="not-prose flex flex-wrap gap-4 my-4">
  <ButtonPrimary>Cliquez-moi</ButtonPrimary>
</div>

## API

| Prop        | Type                                                   | Default     | Description                                   |
| ----------- | ------------------------------------------------------ | ----------- | --------------------------------------------- |
| `children`  | `React.ReactNode`                                      | -           | Contenu du bouton                             |
| `variant`   | `'primary' \| 'secondary' \| 'outline' \| 'ghost'`     | `'primary'` | Variante du bouton                            |
| `size`      | `'sm' \| 'md' \| 'lg'`                                 | `'md'`      | Taille du bouton                              |
| `disabled`  | `boolean`                                              | `false`     | Désactiver le bouton                          |
| `onClick`   | `(event: React.MouseEvent<HTMLButtonElement>) => void` | -           | Fonction appelée lorsque le bouton est cliqué |
| `className` | `string`                                               | `''`        | Classes CSS additionnelles                    |
