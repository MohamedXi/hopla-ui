# À propos de Hopla UI

Hopla UI est une bibliothèque de composants React moderne, accessible et facile à utiliser, conçue pour accélérer le développement d'interfaces utilisateur élégantes et fonctionnelles.

## Caractéristiques principales

- **Accessible** : Tous les composants suivent les normes WAI-ARIA pour garantir l'accessibilité.
- **Personnalisable** : Thèmes et styles facilement adaptables à votre identité visuelle.
- **Typé** : Écrit en TypeScript pour une meilleure expérience de développement.
- **Léger** : Conçu pour être performant et minimiser l'impact sur la taille du bundle.
- **Moderne** : Compatible avec les dernières versions de React et Next.js.

## Philosophie

Hopla UI vise à offrir une expérience de développement agréable tout en produisant des interfaces utilisateur de haute qualité. Notre philosophie repose sur trois piliers :

1. **Simplicité** : API intuitive et documentation claire
2. **Flexibilité** : Adaptable à différents contextes et besoins
3. **Qualité** : Tests rigoureux et attention aux détails
