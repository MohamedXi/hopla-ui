# Hopla UI

Bienvenue dans la documentation de Hopla UI, une bibliothèque de composants React moderne et accessible.

## Composants disponibles

- [Button](/components/button)
- Plus de composants à venir...

## Installation

```bash
npm install hopla-ui
```

## Utilisation

```jsx
import { Button } from 'hopla-ui';

function App() {
  return (
    <Button variant="primary">
      Cliquez-moi
    </Button>
  );
}
```
